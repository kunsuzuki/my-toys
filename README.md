# My Toys Project

## プロジェクト概要
「My Toys」は、ユーザーが写真とテキストを投稿できるシンプルな投稿機能を提供するプラットフォームです。将来的にはブランド選択やいいね機能の追加を予定しています。

### 技術スタック
- **フロントエンド:** Next.js（Reactベース）
- **バックエンド:** Node.js（Express）
- **データベース:** MongoDB Atlas（クラウドベース）
- **仮想環境:** Docker

---

### 必要条件
